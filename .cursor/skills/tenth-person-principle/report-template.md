## 精簡範例（世界盃專頁類型）

參考 [FE-8293](https://innotech.atlassian.net/browse/FE-8293)、[FE-8294](https://innotech.atlassian.net/browse/FE-8294) 的輸出結構：

- §2 多數派：「有 guard、polling、getSportSetting 開關，應該夠了」
- §3 第十人：「新體育全掛但開關仍 true → 不會自動 fallback」
- §4 矩陣：回退機制 / resume 覆蓋 / 用戶提示 等逐項對照
- §5 備案：值班期不發版，靠 OPS 改 setting 或關 feature toggle
