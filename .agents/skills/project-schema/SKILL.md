# Project Schema: pantheon

Pantheon is a tooling-focused repository for Cursor operator workflows. It centralizes commands, rules, scripts, and skills to standardize commit/MR, Jira, and utility automation. The repository acts as a reusable automation platform rather than product runtime code.

## When To Use

Read this skill when:

1. You need to understand which module a file belongs to
2. You are planning changes and need to respect module boundaries
3. You are reviewing whether a new file is placed in the correct directory
4. You are executing `evolve`, `start-task`, or other Operator workflows

## Entry Points

- `.cursor/commands`
- `package.json#scripts`

## Module Details

### Command Layer (`command-layer`)

**職責**：Defines operator-facing command SOPs and execution entrypoints.

**路徑範圍**：
- `.cursor/commands`

**邊界（不應包含）**：Should only describe flow, parameters, and expected outputs; execution logic belongs to scripts.

**依賴模塊**：
- `script-runtime`
- `rule-governance`
- `skill-knowledge`

### Rule Governance (`rule-governance`)

**職責**：Defines global execution policy, reporting standards, and coding/process constraints.

**路徑範圍**：
- `.cursor/rules`

**邊界（不應包含）**：Should not contain executable automation logic or side-effect operations.

**依賴模塊**：
- （無）

### Script Runtime (`script-runtime`)

**職責**：Implements executable automation for CR, Jira, GitLab, notification, and utilities.

**路徑範圍**：
- `.cursor/scripts`

**邊界（不應包含）**：Should not redefine policy documents or long-form command documentation.

**依賴模塊**：
- `rule-governance`
- `root-config`

### Skill Knowledge (`skill-knowledge`)

**職責**：Provides reusable guidance for mounted workflow and external-tool usage decisions.

**路徑範圍**：
- `.cursor/skills`

**邊界（不應包含）**：Should not contain direct script execution or mutable runtime logic.

**依賴模塊**：
- `rule-governance`
- `root-config`

### Root Config (`root-config`)

**職責**：Provides repository metadata, script entry mappings, and adaptation context.

**路徑範圍**：
- `adapt.json`
- `package.json`
- `version.json`
- `README.md`

**邊界（不應包含）**：Should not host command runtime implementation details.

**依賴模塊**：
- （無）
